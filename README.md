# @kazhord/archetype-node-typescript
Typescript archetype for your npm packages.

## Features
- Use Typescript
- ESLint/Prettier
- Write unit tests using jest
- Code coverage
- Comments to documentation with typedoc
- Publish documentation to Github pages

## Scripts

### Build your project
```
npm run build
```

### Run tests
```
npm run test
```

### Generate coverage
```
npm run coverage
```

### Lint project
```
npm run lint
```

### Format project
```
npm run format
```

### Generate doc
```
npm run doc
```

### Publish doc to Github pages
```
npm run publish-doc
```
